Introdução

Este projeto de casa automatizada utiliza o ESP32 e diversos sensores para criar um ambiente inteligente e responsivo. Através de sensores como o DHT11, PIR, e ultrassônico, é possível monitorar e interagir com o ambiente de forma eficiente.

Componentes

ESP32

O ESP32 é o microcontrolador principal do projeto, responsável por controlar os sensores e atuar sobre os dispositivos conectados.

Sensor de Som

Este sensor detecta níveis de som no ambiente, permitindo que a casa responda a comandos de voz ou sons específicos.

Sensor DHT11

O DHT11 é utilizado para medir a temperatura e umidade do ambiente, fornecendo dados essenciais para o controle climático da casa.

Sensor PIR

O sensor PIR detecta movimentos, permitindo que a casa responda à presença de pessoas, por exemplo, acendendo luzes automaticamente.

LED

LEDs são usados como indicadores visuais no sistema, sinalizando o estado de diferentes dispositivos ou sensores.

Chave Táctil

Essa chave permite a interação direta com o sistema, como ligar ou desligar dispositivos manualmente.

Sensor Ultrassônico

Este sensor mede distâncias, podendo ser usado para detectar obstáculos ou medir o nível de líquidos, por exemplo.

Buzzer

O buzzer é utilizado como um alarme sonoro para alertar sobre eventos importantes, como a detecção de movimento ou mudanças bruscas de temperatura.
